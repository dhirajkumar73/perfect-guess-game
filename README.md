# Perfect Guess Game 🎯

A simple number guessing game built using Python.

## 📌 Description
This project generates a random number between 1 and 100.
The user has to guess the correct number with the help of hints like
"Higher number please" or "Lower number please".

## 🚀 Features
- Random number generation
- User input handling
- Higher/Lower hints
- Attempt counter

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Install Python
2. Run the file:
   python main.py
